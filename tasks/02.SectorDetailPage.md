Here is the **Sector Detail Page** content in a clean, minimal table.
Focused only on **Predictions inside a specific sector**.

---

# Sector Detail Page — Main Content Table

| Section           | Purpose          | What the user sees                            |
| ----------------- | ---------------- | --------------------------------------------- |
| Header            | Basic navigation | Logo + Predictions link                       |
| Sector Header     | Show sector info | Sector name, description, markets included    |
| Subtitle          | Page purpose     | “Predictions for this sector”                 |
| Filters           | Narrow results   | Market, horizon, confidence, price range      |
| Sorting           | Order list       | Highest gain, highest confidence, newest      |
| Predictions Table | Main content     | Asset predictions inside this sector          |
| Sector Stats      | Quick overview   | Asset count, prediction count, avg confidence |
| Footer            | Minimal          | Contact + About                               |

---

# Sector Header Table

| Field              | Description                                 |
| ------------------ | ------------------------------------------- |
| Sector Name        | Example: Banking, Energy, Telecom           |
| Description        | Short text (optional)                       |
| Markets Included   | Example: EGX, TASI, ADX                     |
| Asset Count        | Number of assets in this sector             |
| Predictions Count  | Assets with predictions                     |
| Average Confidence | Sector-wide average                         |
| Top Gainer         | Asset with highest expected gain            |
| Top Confidence     | Asset with strongest prediction reliability |

---

# Filters Table

| Filter           | Type          | Purpose                            |
| ---------------- | ------------- | ---------------------------------- |
| Market           | Multi-select  | Narrow by market (EGX, TASI, etc.) |
| Horizon          | Single-select | 7d, 30d, 90d, 180d                 |
| Confidence Range | Slider        | Filter by prediction strength      |
| Expected Gain %  | Slider        | Focus on biggest moves             |
| Price Range      | Slider        | Based on last price                |

---

# Sorting Options Table

| Sorting Option         | Description                |
| ---------------------- | -------------------------- |
| Highest Predicted Gain | Sort by expected growth    |
| Highest Confidence     | Most reliable predictions  |
| Newest Predictions     | Latest updated predictions |
| Alphabetical by Symbol | Symbol A→Z                 |

---

# Predictions Table (Detailed Structure)

| Column          | Description                   |
| --------------- | ----------------------------- |
| Symbol          | Asset symbol                  |
| Name            | Asset full name               |
| Market          | Market code (EGX, TASI)       |
| Last Price      | From `asset_prices`           |
| Predicted Price | From `predicted_asset_prices` |
| Gain %          | (predicted − last) / last     |
| Horizon         | 7d, 30d, 90d, 180d            |
| Confidence      | Prediction confidence         |
| Updated At      | Timestamp of prediction       |
| Action          | “View Details”                |

---

# Sector Statistics (Optional but Useful)

| Metric                   | Description                          |
| ------------------------ | ------------------------------------ |
| Total Sector Assets      | All assets from this sector          |
| Assets With Predictions  | Count of predicted assets            |
| Average Confidence       | Mean confidence score                |
| Sector Volatility        | Based on last price changes          |
| Highest Predicted Gainer | Asset with strongest expected % rise |
| Last Updated             | Latest prediction time               |
