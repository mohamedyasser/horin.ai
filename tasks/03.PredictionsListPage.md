Here is the **Predictions List Page** content in a clean, simple table.
This page is the core page for browsing all predictions.

---

# Predictions List Page — Main Content Table

| Section           | Purpose           | What the user sees                                               |
| ----------------- | ----------------- | ---------------------------------------------------------------- |
| Header            | Basic navigation  | Logo + link to Markets + link to Sectors                         |
| Title             | Show page purpose | “All Predictions”                                                |
| Search            | Quick access      | Search by symbol or name                                         |
| Filters           | Narrow results    | Market, sector, horizon, price range, confidence                 |
| Sorting           | Reorder results   | Highest gain, highest confidence, newest                         |
| Predictions Table | Main content      | Symbol, last price, predicted price, gain %, horizon, confidence |
| Pagination        | Control list size | Page numbers                                                     |
| Footer            | Minimal           | About + Contact                                                  |

---

# Filters Table

| Filter           | Type          | Purpose                     |
| ---------------- | ------------- | --------------------------- |
| Market           | Multi-select  | Focus on specific exchanges |
| Sector           | Multi-select  | Focus on an industry        |
| Horizon          | Single-select | 7d, 30d, 90d, 180d          |
| Confidence Range | Slider        | Filter low/high reliability |
| Expected Gain %  | Slider        | Focus on biggest moves      |
| Price Range      | Slider        | Control price range         |
| Asset Type       | Dropdown      | stock or ETF                |

---

# Sorting Options Table

| Option                 | Description               |
| ---------------------- | ------------------------- |
| Highest Predicted Gain | Biggest upside first      |
| Highest Confidence     | Most reliable predictions |
| Newest Predictions     | Latest updates            |
| Alphabetical           | Symbol A→Z                |
| Market                 | Group by exchange         |

---

# Predictions Table — Detailed Structure

| Column          | Description               |
| --------------- | ------------------------- |
| Symbol          | Stock symbol              |
| Name            | Company name              |
| Market          | Market code               |
| Last Price      | Current price             |
| Predicted Price | Future price              |
| Gain %          | (predicted − last) / last |
| Horizon         | Prediction time frame     |
| Confidence      | Prediction reliability    |
| Updated At      | Prediction timestamp      |
| Action          | Opens Asset Detail Page   |

All rows should be clickable.
