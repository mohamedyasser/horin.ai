# Sector Card / Row Structure

| Field                | Description                         |
| -------------------- | ----------------------------------- |
| Sector Name          | Example: Banking, Energy, Telecom   |
| Description          | Short sector explanation (optional) |
| Asset Count          | Number of companies in this sector  |
| Prediction Count     | How many assets have predictions    |
| Markets Distribution | EGX: x, TASI: y, ADX: z             |
| Action               | Button “View Predictions”           |

# Sectors Listing Table

| Column      | Description                                   |
| ----------- | --------------------------------------------- |
| Sector      | Sector name                                   |
| Assets      | Count of assets linked to this sector         |
