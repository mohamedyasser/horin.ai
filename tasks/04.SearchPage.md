Here is the **Search Page** content in a clean, simple table.
This page helps users quickly find any asset and see its prediction.

---

# Search Page — Main Content Table

| Section            | Purpose            | What the user sees               |
| ------------------ | ------------------ | -------------------------------- |
| Header             | Basic navigation   | Logo + Predictions link          |
| Search Bar         | Main action        | Search by symbol or company name |
| Filters (optional) | Refine results     | Market, sector                   |
| Results Table      | Main content       | List of matches with predictions |
| Empty State        | When nothing found | “No assets match your search”    |
| Footer             | Minimal            | About + Contact                  |

---

# Search Bar Table

| Field        | Description                        |
| ------------ | ---------------------------------- |
| Input        | Text input for symbols and names   |
| Placeholder  | “Search by symbol or company name” |
| Autocomplete | Optional for quicker results       |
| Submit       | Auto-search on typing              |

---

# Results Table — Detailed Structure

| Column          | Description                                |
| --------------- | ------------------------------------------ |
| Symbol          | Asset symbol                               |
| Name            | Company name                               |
| Market          | Market code                                |
| Last Price      | From `asset_prices`                        |
| Predicted Price | From `predicted_asset_prices`              |
| Gain %          | (predicted − last) / last                  |
| Horizon         | Default horizon (30d or highest available) |
| Confidence      | Prediction confidence                      |
| Action          | Click row → Asset Detail Page              |

All rows should be clickable.

---
